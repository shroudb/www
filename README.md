# ShrouDB — www

Marketing site for ShrouDB. Next.js 16 App Router + Tailwind v4 + Radix + CVA.

## Develop

```sh
npm install
npm run dev
```

Open `http://localhost:3000`.

## Structure

```
src/
  app/              App Router pages
  brand/            Design tokens + engine marks (source of truth)
  components/ui/    Reusable primitives
  components/       Site-specific components (header, footer, etc.)
```
